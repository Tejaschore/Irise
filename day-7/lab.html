HTML Classes:

HTML class attribute is used to specify a class for an HTML element. class attribute is often used to point to a class name in a style sheet. It can also be used by a JavaScript to access and manipulate elements with the specific class name.

HTML Id:

id attribute specifies a unique id for an HTML element. The value of the id attribute must be unique within the HTML document.
